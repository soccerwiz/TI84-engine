# PHYSICS

-

CONSTANT/ANGULAR ACCELERATION EQUATIONS:

{
  
Solve for:

> V.initial

> V.final
 
> ^X (Displacement)

> Time              

> Acceleration

GIVEN:

> V.i V.f ^X

> V.i V.f  T

> V.i V.f  A

> V.i  ^X  T

> V.i  ^X  A

> V.i   T  A

> V.f  ^X  T

> V.f  ^X A

> V.f   T  A

}

PROJECTILE MOTION:

{

Solve for:

AT PEAK:

> Time

> ^X (Distance)

> ^Y (Height)

AT GROUND:

> Time

> ^X (Distance)

> Direction (theta) with respect to ground

GIVEN:

> V.initial

> direction(theta) with respect to ground

}
